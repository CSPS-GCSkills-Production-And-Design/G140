Navigation

Back and Next

Instead of a standard navigation bar with Previous and Next buttons, some GCcampus courses have navigation buttons shown as “<” and “>” located at the top and bottom of the right-hand side of the screen. Selecting the “<” or Previous button will take you to the previously viewed page. Selecting the “>” or Next button will bring you to the next page.

French Toggle – Français

Select the “Français” text to switch from the English version of the course to the French version.

Course Home page

Select the Home icon to return to the course Home page.

Help

Select the “Help” icon to access these help topics again.

Toolbox

Select the “Toolbox” icon to download the course content or to access resources to links to useful tools, guidelines, and other information related to performance management.

Sitemap

Select the “Sitemap” icon to access the list of pages in the course.

Navigation Bar

  • Select the navigation bar to navigate among the course.
  • Select a module title to see its related sections.
  • Checkmarks will appear next to modules and sections that you have completed.
